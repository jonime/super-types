# super-types

Really super types for typescript

## Installation


```bash
# NPM
npm install super-types

# YARN
yarn add super-types
```

### `ISODateString`

This is used to add type to [ISO 8601](https://en.wikipedia.org/wiki/ISO_8601) date. Example:

```ts
const date: ISODateString = '2020-12-03';
```
